## What is Open Email?

**Open Email Editor** is a powerful, open-source email builder designed for modern web applications. It leverages [React Email](https://react.email) for reliable rendering and provides a sleek, customizable UI inspired by shadcn/ui.

## Key Features

- **Drag-and-Drop Builder**: Intuitive interface for constructing emails.
- **Layers Panel**: Manage the structure of your email template.
- **Properties Panel**: detailed control over component styles and attributes.
- **Variables**: `{{variableName}}` in content with fallbacks; resolve when generating HTML.
- **JSON Output**: Save templates as JSON and load them back instantly.
- **HTML Export**: Generate production-ready HTML for any email client.
